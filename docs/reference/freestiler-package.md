# freestiler: Create Vector Tiles from Spatial Data

Create vector tile archives in 'PMTiles' format from 'sf' spatial data
frames. Supports 'Mapbox Vector Tile' ('MVT') and 'MapLibre Tile'
('MLT') output formats. Uses a 'Rust' backend via 'extendr' for fast,
in-memory tiling with zero external system dependencies.
